Here is a list of options:
```
Commands:
"-cool" - Embed and send problem to #cool-problems
"-marathon" - Embed and send problem to #marathon-problems

Flags:
"-title" - Title for the problem.(may include problem no.)
"-problem" - Statement or description for the problem.
"-hint" - (optional) Hint for the problem. Should be spoilered("|| spoiler ||")
"-source" - (optional) Source of the problem(spoiler)
"-difficulty" - (optional) Estimated difficulty of the problem(spoiler)
```
